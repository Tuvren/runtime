# Constitutional Archive

Everything under `constitution/archived/` is retained for audit and historical
context only.

- Archived files are not part of the live constitutional authority chain.
- Archived files may describe prior closure, readiness, inventory, or planning
  state that no longer governs current execution.
- Promote or cite an archived artifact as live authority only if one of the
  four live constitutional documents explicitly re-adopts it in a future change.
